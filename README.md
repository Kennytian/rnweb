## React Native APP and React Native Web in ONE

### Step 0
```
git clone https://github.com/Kennytian/rnweb.git
```

### Step 1
```
cd rnweb && yarn
```

### Step 2
```
yarn start
```

### Step 3
> run native app

```
yarn start android
```

or

```
yarn start ios
```

### Step 4
> run web app

```
$ yarn run web
```
