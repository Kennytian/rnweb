# React Native APP 和 React Native Web 项目

启动项目

### Step 1
```
yarn
```

### Step 2
```
yarn start
```

### Step 3
> run native app

```
yarn start android
```

or

```
yarn start ios
```

### Step 4
> run web app

```
$ yarn run web
```
